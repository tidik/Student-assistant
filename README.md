# 基于PHP开发的请假系统（23毕设）

> 简单实现电子假条的小功能

## 项目组成

| Component      | Description |
| ----------- | ----------- |
|   PHP+Layui   | 控制器       |
| MySQL   | 数据库        |
| Android   | 采集数据        |


![流程图](https://img2.imgtp.com/2024/03/27/AsFxILVp.png)

## 功能列表
 * 批准/拒绝 支持邮件通知
 * 后端可以主动发送邮件给用户

## 注意事项

* 修改 core/config.php 中相关信息（MySQL和SMTP信息）
* 默认用户名 admin 
* 默认密码 123456
